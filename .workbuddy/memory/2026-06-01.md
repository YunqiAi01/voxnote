## VoxNote - 语音速记+一键整理

### 文件结构
```
src/
├── components/
│   └── RecordButton.tsx    # 圆形录音按钮组件（MediaRecorder + 脉冲动画）
├── app/
│   ├── api/
│   │   ├── transcribe/
│   │   │   └── route.ts    # Deepgram 语音转写 API
│   │   └── organize/
│   │       └── route.ts    # OpenAI (gpt-4o-mini) 笔记整理 API
│   ├── globals.css         # Tailwind v4 + shimmer动画
│   ├── layout.tsx          # 根布局
│   └── page.tsx            # 主页面（5状态流转：idle→recorded→transcribing→transcribed→organizing）
```

### 技术要点
- **录音**: 浏览器 MediaRecorder API，webm 格式，红色脉冲动画
- **转写**: POST 音频 blob → `/api/transcribe` → Deepgram API (zh-CN, smart_format)
- **整理**: POST 文本 → `/api/organize` → OpenAI gpt-4o-mini + 精心调试的 system prompt → Markdown 结构（核心观点/关键细节/行动项）
- **UI 状态机**: idle → recorded → transcribing → transcribed → organizing，左右对比原始稿与结构化笔记
- **环境变量**: DEEPGRAM_API_KEY, OPENAI_API_KEY 需在 .env.local 中配置
- **修复**: styled-jsx 在 Next.js 16 + Turbopack 下有兼容问题，改用 globals.css 中的 @keyframes
